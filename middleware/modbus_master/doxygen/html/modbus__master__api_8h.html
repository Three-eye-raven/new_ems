同步从设备所有的可读信息

参数
pChannel通讯通道
slaveIdx从设备索引号
返回
ret的内容按照bit位来表示同步结果，若同步无异常，ret = 0； bit0:pCtx为空； bit1:slaveIdx越界 bit2:未绑定协议表 bit3:协议表不完整，缺失读取内容 bit4:通道未初始化 bit5:协议表内容不一致 bit6:读取失败
